# Epsilon Delta Definition

<MathBlock type="theorem" title="Epsilon Delta Definition">
    Let $D \subseteq  \mathbb{R}$, $f : D \to \mathbb{R}$, and $a \in D$. 

    Then $\forall \varepsilon > 0 $,  $\exists \delta > 0$ such that $\forall x \in D: 
        |x - a| < \delta$ $\Rightarrow$  $|f(x) - f(a)| < \varepsilon$
</MathBlock>
